# 📦 C++ Object-Oriented Programming (OOP) Exercises

This repository contains my source code, exercises from the Object-Oriented Programming (OOP) course. It serves as a personal learning journal to demonstrate my understanding of core software design principles and C++ programming.

---

## 🚀 Key OOP Principles Applied

In this repository, I focus on implementing the 2 pillars of Object-Oriented Programming:

*   **Encapsulation (Đóng gói):** Designing classes with private attributes and public getters/setters to protect and hide data.
*   **Inheritance (Kế thừa):** Reusing code and building hierarchical structures (Single, Multiple, and Hierarchical inheritance).

## 📝 How to Run the Projects

### Option 1: Using IDEs (Dev-C++ / Visual Studio Code / Visual Studio)
1. Download or Clone this repository to your local machine.
2. Open your preferred IDE (e.g., **Dev-C++**).
3. Open the file `main.cpp` (or the specific source file inside the `./LTHuongDoiTuong/` folder).
4. Press **F11** (in Dev-C++) to Compile and Run the program directly.

### Option 2: Using Command Line (GCC Compiler)
1. Clone this repository:
   ```bash
   git clone [https://github.com/Duynv476/cpp-oop-principles.git](https://github.com/Duynv476/cpp-oop-principles.git)
2. Navigate to the source folder:
   cd cpp-oop-principles/LTHuongDoiTuong
3. Compile the C++ file
   g++ main.cpp -o program
