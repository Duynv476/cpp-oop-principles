# 📦 C++ Object-Oriented Programming (OOP) Exercises

This repository contains my source code, exercises, and projects from the Object-Oriented Programming (OOP) course. It serves as a personal learning journal to demonstrate my understanding of core software design principles and C++ programming.

---

## 🚀 Key OOP Principles Applied

In this repository, I focus on implementing the 4 pillars of Object-Oriented Programming:

*   **Encapsulation (Đóng gói):** Designing classes with private attributes and public getters/setters to protect and hide data.
*   **Inheritance (Kế thừa):** Reusing code and building hierarchical structures (Single, Multiple, and Hierarchical inheritance).
